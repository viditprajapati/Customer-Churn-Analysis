📌 Overview

This project presents a Customer Churn Analysis Dashboard built using Power BI to analyze customer behavior and identify key factors contributing to churn.

The dashboard helps businesses understand why customers leave, enabling them to take proactive retention strategies.

🎯 Objectives
Analyze overall churn rate and customer distribution
Identify factors affecting churn (contract, services, payment methods)
Study customer demographics and behavior
Understand the impact of tenure and services on churn
🛠️ Tech Stack
Power BI Desktop – Dashboard creation
Power Query – Data cleaning & transformation
DAX – Calculations for churn rate, KPIs
Data Modeling – Relationships between customer attributes


📂 Data Description
The dataset includes customer-related information such as:

Customer ID, Gender
Contract Type (Monthly, Yearly, etc.)
Internet Service Type (DSL, Fiber, None)
Payment Method
Monthly Charges & Total Charges
Tenure (customer duration)
Churn Status (Yes/No)

As shown on page 1, the dataset contains 7043 customers with 26.54% churn rate.

Customers with month-to-month contracts have higher churn
Fiber optic users show higher churn rates
Lack of tech support increases churn probability
Customers with longer tenure are less likely to churn
Payment methods like electronic check show higher churn
